# Snake

Small dependency-free Snake game.

## Run

Open `index.html` in a browser, or serve the folder with any static file server and open the repo root.

## Manual checklist

- Start movement with arrow keys or `WASD`.
- Confirm the snake grows and score increments after eating food.
- Confirm reverse direction is blocked.
- Confirm hitting a wall or the snake body ends the game.
- Confirm `Pause` / `Resume` and `Restart` work.
- Confirm touch controls work on narrow/mobile screens.
